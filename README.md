# 🦁 Simha — All Shops, One Place

A local e-commerce platform connecting Nepal's shops with customers. Built with Firebase, ImageBB, and WhatsApp integration.

## 🚀 Features

- **Multi-shop marketplace** — Browse shops by category, district, or search
- **Cart with shop grouping** — Order from multiple shops, each with their own checkout
- **WhatsApp order integration** — Orders sent directly to shop owners via WhatsApp
- **Shop owner dashboard** — Manage products, view orders, track analytics
- **Admin panel** — Create shops, manage users, send notifications
- **Push notifications** — Real-time alerts via Firebase Cloud Messaging
- **Image upload** — Free image hosting via ImageBB API
- **PWA ready** — Install on mobile devices

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML, CSS, JavaScript
- **Backend**: Firebase (Auth, Firestore, Hosting, Messaging)
- **Image Storage**: ImageBB (free unlimited storage)
- **Notifications**: Firebase Cloud Messaging + EmailJS
- **Deployment**: Firebase Hosting
